# Restful API
Basic CRUD with Gin, GORM, and PostgreSQL

## Installation
- PostgreSQL Driver ([lib/pq](https://github.com/lib/pq))
- [Gin Framework](https://github.com/gin-gonic/gin)

## CRUD Features
- `POST`
- `GET`
- `PUT`
- `DELETE`

## Port
- `http://localhost:8080/api`

## Run
```bash
go run main.go
